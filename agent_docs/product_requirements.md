# Product Requirements

## Core User Journeys
1. **App Launch:** Display the Hero Card in a highly polished, spatial UI.
2. **Card Emulation:** Enable NFC Host Card Emulation (HCE) to mimic physical DESFire library cards when tapped against library terminals.
3. **Security:** Require Biometric authentication for sensitive workflows. Store payloads securely using EncryptedSharedPreferences.
4. **Spatial UI:** Maintain physical, multi-layered interfaces with subtle background animations and fluid navigation.
